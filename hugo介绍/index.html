建立hugo个人博客的步骤

1.在本机上搭建hugo

  • hugo new site myblog ：创建一个名为myblog的站点
  • git clone 、、、下载hugo主题
  • hugo new ..md 建立一个md文档，最后变成后来的html
  • hugo server -t 主题名 –buildDrafts 以相应主题启动hugo

2.推向GitHub，以公网访问

  • hugo –theme=主题名 –baseUrl=GitHub上建立的仓库名(要求为你的名称+.github.io，比如ysumgl.github.io)这回在本地生成一个public文件夹，即把写的文章弄成HTML版，例如hugo –theme=m10c –baseUrl=“https://ysumgl.github.io/"
  • 进入public文件夹，
    • git init
    • git add .
    • git commit -m “一些描述”
    • git remote add origin https://github.com/ysuMgl/ysumgl.github.io.git
    • git push -u origin master

进行这几部操作，将项目推到GitHub上去，就可以通过公网访问了

3.写过文章后，在GitHub中更新

  • git add .
  • git commit -m “一些描述”
  • git pull
  • git push origin master
